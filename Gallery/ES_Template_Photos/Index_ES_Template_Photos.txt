Nice Minecraft Screenshots
A set of random screenshots I took from the hit game Minecraft: Java Edition by Swedish Game Developer Mojang Studios as placeholder images to test this page's system. This layout is inspired by Instagram's simple photo gallery layout. The system relies on a text file that contains all the links of the individual placeholder and full quality images and uses loops to repeatedly create thumbnails and previews of each photo. If done right, a nice simple gallery will be created from unconventional means.
